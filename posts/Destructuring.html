What is Destructuring?

Destructuring was introduced in ES6 and is a very powerful JavaScript feature.

Consider the following object:

const user = {
  name: "Bob",
  age: 20,
}

Destructuring allows us to extract values directly from this object and assign them to variables:

const { name, age } = user;

As opposed to: 

const name = user.name; 
const age = user.age;
